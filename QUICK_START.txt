 INSTALLATION EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════════════════

ÉTAPE 1 : Accédez à la page d'installation
──────────────────────────────────────────
URL : http://localhost:8080/Teranga_azur/install-payment-system.php

Cette page va :
  ✓ Vérifier tous les fichiers
  ✓ Créer automatiquement la table de base de données
  ✓ Afficher le statut de chaque composant

ÉTAPE 2 : Attendez la confirmation
──────────────────────────────────
Vous devriez voir un message vert : "Excellent! Tous les composants sont..."

Si quelque chose échoue :
  ✗ Vérifiez que XAMPP (MySQL) est en cours d'exécution
  ✗ Vérifiez que le port 8080 n'est pas bloqué
  ✗ Vérifiez que le fichier php.ini permet les extensions PDO


ÉTAPE 3 : Commencez à tester
─────────────────────────────
Cliquez sur le bouton "Tester avec les Villas" ou "Tester avec les Activités"

 DÉPANNAGE RAPIDE
═════════════════════════════════════════════════════════════════════════════

 Erreur : "La table reservations n'existe pas"
→ Visitez install-payment-system.php pour créer la table

 Erreur : "Connexion à la base de données impossible"
→ Vérifiez que MySQL est démarré dans XAMPP
→ Port doit être 3307 (pas 3306)

 Les boutons "Réserver" ne font rien
→ Vérifiez que villas.php et activites.php ont les bons liens
→ Recherchez "checkout.php" dans les fichiers

 La page est blanche ou 404
→ Vérifiez l'URL : http://localhost:8080/Teranga_azur/...
→ Assurez-vous que XAMPP Apache s'exécute sur port 8080

 Les prix ne s'affichent pas
→ Ouvrez la console développeur (F12)
→ Vérifiez les erreurs JavaScript
→ Assurez-vous que les dates sont sélectionnées

 NUMÉROS DE TEST FICTIFS

Autres champs :

   Expiration        : Format MM/YY (ex: 12/28, 06/26, etc.)
   CVC               : N'importe quel 3 chiffres (ex: 123, 456, 789)

 QUESTIONS FRÉQUENTES
═════════════════════════════════════════════════════════════════════════════

Q: Est-ce un vrai système de paiement?
R: NON - C'est un système FICTIF pour démonstration (projet de classe)

Q: Où sont sauvegardées les réservations?
R: Table "reservations" dans la base de données MySQL

Q: Puis-je modifier les textes et styles?
R: OUI - Tous les fichiers sont modifiables

Q: Comment ajouter plus de champs au formulaire?
R: Éditez paiement.php et confirmation.php

Q: Peut-on envoyer des emails de confirmation?
R: Oui, mais il faudra ajouter PHPMailer
